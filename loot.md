# loot

loot has:
    gameobject
    portrait
    value
    description
    name
    categories
    optional modifiers
        extra value?

## NPCs drop loot

NPCs have a loot collection

loot collection is a collection of loot sets with probability:
    1x valuable with prob 0.5
    2x low value with prob 0.7
